Новый чистый Telegram бот для Maxxpharm

## Особенности

- ✅ Прямая проверка ID администратора
- ✅ Минимальный код без зависимостей

## Установка

1. Скопируйте репозиторий
2. Установите BOT_TOKEN в переменных окружения Render
3. Разверните на Render

Отправьте команду `/start` боту @solimfarm_bot
